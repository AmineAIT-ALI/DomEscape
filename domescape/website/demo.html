DÉMONSTRATION

DomEscape Lab 01

Un laboratoire sécurisé où quatre capteurs Z-Wave valident quatre énigmes physiques. L'escape game est le proof of concept — pas le produit.

SCÉNARIO

Réactiver le système.

Un groupe de joueurs doit réactiver le système de sécurité d'un laboratoire dans la bonne séquence. Chaque action physique déclenche un vrai événement capteur. Le moteur valide l'action et réagit.

ÉTAPES

Quatre étapes. Quatre capteurs. Une séquence.

01
Démarrage système
BUTTON_PRESS

Appuyer une fois sur le bouton du bureau pour initialiser le système. L'action correcte active la prise murale et passe à l'étape 2.

Périphérique
Fibaro Button FGPB-101
Type capteur
button
Actions déclenchées
PLUG_ON + LCD "Système en ligne."
02
Porte secrète
DOOR_OPEN

Ouvrir la porte sécurisée pour accéder à la zone restreinte. Le capteur de porte détecte l'ouverture et valide l'étape.

Périphérique
Fibaro Door Sensor FGDW-002
Type capteur
door_sensor
Actions déclenchées
LCD "Accès autorisé."
03
Scan de mouvement
MOTION_DETECTED

Traverser la zone de détection pour compléter le scan biométrique. Le capteur de mouvement se déclenche à la présence d'une personne.

Périphérique
Aeotec Multisensor 6
Type capteur
motion_sensor
Actions déclenchées
PLUG_ON + LCD "Scan validé."
04
Code final
BUTTON_DOUBLE_PRESS

Appuyer deux fois sur le bouton Fibaro pour sceller le laboratoire. Un simple clic ou un triple clic incrémente le compteur d'erreurs sans avancer.

Périphérique
Fibaro Button FGPB-101
Type capteur
button
Actions déclenchées
PLUG_ON + LCD "ESCAPE SUCCESSFUL !"

CE QUE ÇA PROUVE

Trois propriétés, une seule démo.

Interface physique avant tout

Pas d'écran tactile. Pas de clavier. Les joueurs interagissent avec de vrais objets : boutons, portes, capteurs de mouvement.

Feedback en moins d'une seconde

Du déclenchement capteur au message LCD et à la réaction de la prise : moins d'une seconde. Événement → Moteur → Actionneur.

Infiniment reconfigurable

Changer de scénario ne nécessite aucune modification du code. Chaque étape, capteur et réaction est configuré dans la base de données.
